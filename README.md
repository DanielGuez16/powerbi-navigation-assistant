# Power BI Navigation Assistant

A small RAG (retrieval-augmented generation) assistant that helps users
**discover the right Power BI report** in a large catalog by asking a question
in natural language.

The app indexes every report's `(workspace, report, page)` tuple as a vector
embedding, retrieves the top-k semantically closest entries to the user's
question, and feeds them to an LLM that returns a ranked list of suggestions
with clickable links.

Built with **Streamlit**, **OpenAI** (LLM + embeddings), **scikit-learn**
(cosine similarity) and the **Microsoft Graph / Power BI REST APIs** for
optional enterprise integrations.

---

## Features

- **Natural-language search** over a Power BI report catalog.
- **Per-team scoping**: results are filtered to the workspaces the user's team
  is allowed to see (`data/team_workspace_mapping.json`).
- **Glossary injection**: a per-team Excel glossary is included in the LLM
  context so domain-specific acronyms are understood.
- **Pluggable storage**: glossaries live as local Excel files by default, and
  can transparently switch to SharePoint Online via a single env var.
- **Pluggable LLM backend**: works out-of-the-box with OpenAI; swap to Azure
  OpenAI or any OpenAI-compatible endpoint by setting `OPENAI_BASE_URL`.

## Quick start

### 1. Clone & install

```bash
git clone <this-repo>
cd <this-repo>

python -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

### 2. Configure

```bash
cp .env.example .env
# edit .env and set OPENAI_API_KEY=sk-...
```

The minimum to run the demo is **just an OpenAI API key**. Everything else is
optional and gracefully falls back to local sample data.

### 3. Build the embeddings index

This crunches `data/metadata.json` (sample data is bundled) into a numpy
embeddings array:

```bash
python embeddings_metadata.py
```

You should see something like:

```
✅ 39 embeddings generated and saved to .../data.
```

### 4. Run the app

```bash
streamlit run app_pbi.py
```

Open <http://localhost:8501>, pick a team (Finance, Engineering, Data
Management or Other), and ask a question — e.g. *"Where can I find the
liquidity coverage ratio?"* or *"Show me reports about cash flow"*.

## Replacing the sample data

The bundled `data/metadata.json` is illustrative. To point at your own Power BI
tenant:

1. Register an Azure AD app with the
   [Power BI service-principal permissions](https://learn.microsoft.com/en-us/power-bi/enterprise/service-premium-service-principal).
2. Fill in `POWERBI_TENANT_ID`, `POWERBI_CLIENT_ID`, `POWERBI_CLIENT_SECRET` in
   your `.env`.
3. Crawl every accessible workspace:
   ```bash
   python powerbi_connector.py
   ```
4. Re-build the embeddings index:
   ```bash
   python embeddings_metadata.py
   ```

Edit `data/team_workspace_mapping.json` to map each team to the workspaces it
should see, and `prompts.py` if you want to tweak the per-team system prompt.

## Switching glossaries to SharePoint

Set in `.env`:

```bash
USE_SHAREPOINT=true
SHAREPOINT_TENANT_ID=...
SHAREPOINT_CLIENT_ID=...
SHAREPOINT_CLIENT_SECRET=...
SHAREPOINT_SITE_HOSTNAME=yourtenant.sharepoint.com
SHAREPOINT_SITE_RELATIVE_PATH=/sites/YourSite
```

`glossary_loader.read_glossary()` and `save_glossary()` will then transparently
hit the SharePoint Graph API instead of the local filesystem.

## Configuration reference

| Variable                       | Required | Default                  | Purpose                                              |
| ------------------------------ | -------- | ------------------------ | ---------------------------------------------------- |
| `OPENAI_API_KEY`               | yes      | —                        | Chat & embeddings                                    |
| `OPENAI_BASE_URL`              | no       | OpenAI public endpoint   | Azure OpenAI / proxy                                 |
| `LLM_MODEL`                    | no       | `gpt-4o-mini`            | Default chat model                                   |
| `EMBEDDING_MODEL`              | no       | `text-embedding-3-small` | Default embedding model                              |
| `USE_SHAREPOINT`               | no       | `false`                  | Toggle glossary backend                              |
| `SHAREPOINT_*`                 | if SP    | —                        | Microsoft Graph credentials                          |
| `POWERBI_*`                    | if crawl | —                        | Power BI service principal                           |
| `HTTP_PROXY` / `HTTPS_PROXY`   | no       | —                        | Corporate proxy                                      |
